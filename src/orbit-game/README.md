# Orbit game (kioku/orbit)

Upstream: [kioku/orbit](https://github.com/kioku/orbit) — MIT License.

Vendored `orbit.ts` + `orbit.css` with Petralian patches:

- Endless mode only (no survival timer / score target)
- Difficulty: +1% enemy speed every 5 seconds
- Container-sized canvas (not full window)
- Petralian leaderboard callback + share URL
- PostHog removed; spacebar scroll prevented in React shell
